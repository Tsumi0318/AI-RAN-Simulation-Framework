# Mark4输出文件说明

## 第3节验证指标

| PDF要求 | 文件 | 内容 |
|---|---|---|
| 四个基准 | `algorithm_comparison.csv` | PSNE、All-Local、All-Offload、Random、Greedy |
| 收敛与稳定性 | `convergence_trace.csv` | 每轮势函数、卸载数、总代价、VRAM负载、策略变化 |
| 均衡策略 | `equilibrium_strategy_s_star.csv` | 100个节点最终$s^*$及语义倍率 |
| 帕累托与总代价 | `pareto_front.csv` | $K=0\ldots100$排序候选与非支配标记 |
| 显存违规率 | `memory_violation_sweep.csv` | $N=30\ldots200$、五算法、30次固定种子 |
| LLM开销 | `llm_coordination_overhead.csv` | 调用、缓存、Token、平均/P95时延 |
| LLM逐轮反馈 | `llm_feedback_events.csv` | 输入状态、数值回复、语义警告和公式校验 |

| 文件 | 内容 |
